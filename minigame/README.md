# 八卦立方体 - 微信小游戏

基于易经八卦的微信小游戏，Canvas 2D 渲染。

## 项目结构

```
minigame/
├── game.js              # 主程序（完整独立版本）
├── game.json            # 小游戏配置
├── project.config.json  # 项目配置
└── README.md
```

## 功能

- 八卦立方体可视化（8个顶点代表八卦）
- 火柴人行走动画
- 点击顶点切换宫位视角
- 场景元素（树、草、花）

## 使用方法

1. 在微信开发者工具中打开 `minigame` 目录
2. 点击编译即可运行

## 交互

- **点击顶点**: 切换到对应宫位视角
- 八宫: 乾、坤、震、巽、坎、离、艮、兑
